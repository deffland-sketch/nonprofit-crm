Import Tips
  • Import donors first if you're importing both donors and donations
  • The first row of your Excel file should contain column headers
  • Duplicate donors (same name or email) will be skipped
  • Donations will automatically create donors if they don't exist
  • Supported date formats: YYYY-MM-DD or MM/DD/YYYY
  • Amount can include $ and commas (e.g., "$1,000.00")
